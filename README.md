# DotDecoder (BIP-39 Binary Decoder)

DotDecoder는 BIP-39 시드 구문을 이진수(Binary) 점 패턴으로 변환하거나, 점 패턴을 단어로 변환해주는 도구입니다.
[TINSB](https://store.btcmap.kr/stores/TINSB/) 금속판 사용자를 위해 1-based index(1~2048) 체계를 지원합니다.

## ✨ 주요 기능 (Features)
- **양방향 변환**: Dot ↔ Word 실시간 변환
- **오프라인 강제**: 네트워크 연결 감지 시 앱 사용 차단
- **보안 기능**: 클립보드 복사 방지, 캡처 방지, 자동 데이터 파기
- **편의성**: PWA 지원, 다크 모드, 반응형 디자인
